# Pizza App

Welcome to the tiniest pizza studio! Run the generator below to dream up unexpected pies for your next dinner party.

```bash
python pizza_generator.py --seed 42 5
```

Each run produces a set of playful combinations that mix crusts, sauces, cheeses, and toppings from a curated list. Adjust the seed for reproducible results or omit it to let fate choose the flavors.
